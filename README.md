# CGS Games

Website for Card Game Simulator (CGS) users to share their games.

Built with Next.js and Firebase.

## Instructions

**First**: `npm install`

**Format**: `npx prettier . --write`

**Audit**: `npm audit fix`

**Lint**: `npm run lint`

**Build**: `npm run build`

**Start**: `npm run start`

**Deploy**: `firebase deploy`
